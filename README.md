# JavaScript Code Quiz

## About the quiz
This JavaScript quiz is made up of currently four questions centered around JavaScript and the user has 60 seconds to answer them.

The final score is determine on how fast the user finished the quiz and how accurate his answers are. If the user chooses the wrong answer then they lose 10 points on their final score.
